Course Details

The values of the following SCORM data elements can be edited through Captivate by clicking Edit > Preferences > Reporting > Configure. The Manifest dialog opens where you can specify these values.

Course_id

The ID of the course that contains the SCORM activity. Edit the Identifier field in the Course area to update this section.

Course_title

The name of the course that contains the SCORM activity. Edit the Title field in the Course area to update this section.

Sco_id

The ID of the SCORM course. Edit the Identifier field in the SCO area to update this section.

Sco_title

The title of the SCORM course. Edit the Title field in the SCO area to update this section.

The values of the following SCORM data elements are populated at run-time depending on the learner's performance.

Attempts

number of attempts made by the learner.

Suspended

indicates whether the course is suspended or not.

Objective_id

The ID of the quiz to which the question slide belongs. In a master quiz containing multiple quizzes, an objective ID helps identify the quiz to which the question belongs

LMS Result view help

Runtime Data

Completion_status

Indicates whether the learner has completed the course objectives. This can be set within Captivate in Edit > Preferences > Reporting in the Completion criteria area

Success_status

Indicates whether the learner has mastered the course or not. This can be set within Captivate in Edit > Preferences > Reporting in the Success criteria area.

Entry

indicates whether the learner is starting the course for the first time or is resuming a prior attempt.

Exit

indicates how the learner is exiting the course

Location

If the course is suspended, this indicates the last slide that the learner viewed before exiting the course.

Score_raw

Indicates the score that the learner earned on an assessment in the course

Score_max

Indicates the maximum possible score that a learner can earn in an assessment

Score_scaled

Indicates the number that reflects the performance of the learner on a scale that is defined by the organization

Interaction Data

Sl. No

Serial Number

ID

The question identifier. This is used to associate the question to a database or master list of questions. You can edit this ID in the property Inspector of the question slide. You can also add a prefix to the ID in the Edit > Preferences > Quiz Settings >Interaction ID Prefix text field. Use this field if you want to customize the generated interaction IDs by prefixing them with the assigned characters

Type

Type of question, for example, multiple-choice, true/false, matching, and so on.

Timestamp

The time when this item was presented to the learner

Correct responses

Defines an array of correct responses to be presented. The array is a list of patterns that correspond to the type of interaction.

Weighting

Weight assigned to the question when calculating score

Learner response

What the learner actually answered

Result

Whether the learner's response was correct or not

Latency

The time it takes for the learner to respond to the question

Description

A description of the interaction. This might be the question presented or a description of the task given to the learner
